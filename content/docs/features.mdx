# Features

SIRC includes powerful features designed for modern IRC users.

## Auto-Reconnect

SIRC automatically reconnects when your connection drops:

- Exponential backoff prevents server flooding
- Automatically rejoins all channels
- Preserves your nickname when possible
- Seamless recovery without manual intervention

## NickServ Authentication

Automatic authentication with NickServ:

- Configure passwords per network
- Auto-identify on connect
- Secure credential storage
- Support for multiple authentication methods

## Inline Image Preview

View images directly in chat:

- Automatic image detection from URLs
- Lazy loading for performance
- Preview common image formats (PNG, JPG, GIF, WebP)
- Click to view full size

## Visual WHOIS

Professional WHOIS information display:

- Organized sections for easy reading
- Shows user info, channels, idle time
- Server information
- Connection details

## TLS Encryption

All connections use enterprise-grade encryption:

- TLS 1.2+ support
- Certificate validation
- Secure by default
- No plaintext passwords

## XDCC Support

Built-in XDCC file transfer:

- Download files via XDCC
- Queue management
- Progress tracking
- Resume support for interrupted downloads

## Multi-Server Support

Connect to multiple servers simultaneously:

- Separate windows for each server
- Quick server switching
- Server-specific settings
- Custom port and SSL configuration
