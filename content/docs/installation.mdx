# Installation

SIRC is available for Windows and macOS. Follow the instructions below for your platform.

## Windows

1. Download the Windows installer (.exe) from the [downloads page](/)
2. Run the installer
3. Follow the installation wizard
4. Launch SIRC from your Start menu

## macOS

1. Download the macOS installer (.dmg) from the [downloads page](/)
2. Open the .dmg file
3. Drag SIRC to your Applications folder
4. Launch SIRC from Applications

## System Requirements

### Windows
- Windows 10 or later
- 200 MB free disk space
- Internet connection

### macOS
- macOS 11 (Big Sur) or later
- 200 MB free disk space
- Internet connection

## First Launch

When you first launch SIRC:

1. The app will ask for necessary permissions
2. You'll see the main window with the server connection dialog
3. Add your first IRC server to get started

## Updates

SIRC will automatically check for updates and notify you when a new version is available.
