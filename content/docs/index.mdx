# Welcome to SIRC

SIRC is a modern IRC client built for the decentralized era. This documentation will help you get started with using SIRC and understanding its features.

## Features

- **Smart Auto-Reconnect**: Automatic reconnection with exponential backoff
- **NickServ Authentication**: Automatic identification on connect
- **Inline Image Preview**: Images display directly in chat
- **Visual WHOIS**: Professional WHOIS display
- **Encrypted Connections**: Enterprise-grade TLS encryption
- **XDCC Support**: Built-in XDCC file transfer support
